<p>注意:框架本身功能未动，只修复了微信代付功能

快捷接入
平台提供简单的API对接文档方便一切接口快速接入！
代付通道
本平台支持微信支付宝发方式，帮助个人和企业完成下发功能。
在线汇款
为了完全解决用户出款需求，本平台支持一键出款。
